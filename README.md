# ActivityPub Server 

This implementation aims to be fully compliant with the ActivityPub server spec.

## Stack:
* FastAPI
* MongoDB
* Taskiq (Redis)

## Env Vars:
* MONGO_URL
* REDIS_URL
* SERVER_PRIVATE_KEY
